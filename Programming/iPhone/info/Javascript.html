iPhone SpiderMonkey Port.

To run Javascript, launch MobileTerminal, and type “js”.
Some sample code you can try is: 

js> x = 3
3
js> print ( x + 4 )
7
(Don’t type the “js>”, that’s from the interpreter).
Use ^D to exit the interpreter.
